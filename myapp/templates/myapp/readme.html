Django-app: Running-diary

Django-application for running statistics.

Site structure

For unauthorized user:

  • ReadMe
    • Data on the installation and use of the application
  • Login
    • Authorization to enter the site
  • Sinup
    • Registration on the site

For authorized user:

  • ReadMe
    • Data on the installation and use of the application
  • Add data
    • Form for adding new data.
  • Accepted data
    • 1. Date and time.
    • 2. Distance (in meters).
    • 3. Duration (in minutes).
  • Data list
    • A list of all activities. The speed according to the results of the recording is also indicated. Separately, the average speed for all records. There is a filter records by date.
  • Statistic
    • Statistics of all records grouped by week for last year - week number, total amount of entries, total distance, total duration and weekly average speed.
  • Logout
    • Stop logging in as a registered user.

Technologies and Libraries

  • Python3.7
  • Django==2.2.2
  • django-tables2==2.0.6
  • HTML/CSS
  • SQLite3

Prerequisites

Needed to be installed Python3. If not installed, download the link https://www.python.org/ and install.

Deployment

Run command:

pip3 install -r requirements.txt

Database Setup

python manage.py migrate

Start app

python manage.py runserver
